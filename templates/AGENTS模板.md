## 一、本项目接入了"东合施工队" skill

凡是接入本项目的 agent，进入工作前必须按 skill 流程：

- Codex：`~/.codex/skills/donghe-construction-team/SKILL.md`
- Cursor：`~/.cursor/skills-cursor/donghe-construction-team/SKILL.md`

**首次进入** → 检查本文件是否含上方 `donghe-construction-team: initialized` 标识。
- 若无 → 走 `playbooks/00-接活契约.md` 的"首次进入子流程"完成踏勘后再创建本文件
- 若有 → 走"已接入项目快速对齐"

**关键约定**：
- 完工前先看 `docs/工地踏勘报告.md` + `docs/路线图.md` + `docs/下一阶段工作.md` + `docs/项目知识库.md` + `docs/工地禁忌.md` + `docs/施工日志/最近 3 天`
- M / L 级工作必须在 `docs/任务卡/` 落盘任务卡
- 完工必带真实运行时证据
- 当日工作结束前必写当日施工日志
- 多队模式下额外读 `docs/施工队-临界占用.md` + `docs/施工日志/<今天>/`
