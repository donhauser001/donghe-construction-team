## 十五、下一步建议

### 推荐 · 审临时分支

```bash
git fetch
git diff origin/<base>..origin/auto/YYYY-MM-DD-HHMM
```

### 合并

```bash
git switch <base>
git merge --squash auto/YYYY-MM-DD-HHMM
git commit -m "[auto-batch] 自动施工成果"
```

### 挑选

```bash
git switch <base>
git cherry-pick <commit hash>
```

### 废弃

```bash
git branch -D auto/YYYY-MM-DD-HHMM
git push origin --delete auto/YYYY-MM-DD-HHMM
```
